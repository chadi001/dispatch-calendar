Dispatch Guide
Quick onboarding for dispatchers and new users.
1) Sign in
Use Workspace Access at top-right.
2) Filter
Use Administrative Region + Client on the left, and Tech/Group filters on the right.
3) Plan
Generate plan -> Optimize routes -> Compare scenario -> Apply scenario.
4) Multi-tech jobs
Enable Multi-Tech, open day details, add extra techs, set group, then Save team.
5) Area deployment
Enable Area Rules and define temporary region rules (e.g., Quebec area surge period).
6) Database sync and reset
Apply scenario auto-syncs DB. Reset restores backup first, then fallback to imported state.
